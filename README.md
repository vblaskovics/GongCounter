# Go counter app demo

## Install

### Backend

1. Download and install Go

- https://golang.org/

2. Set PATH

- https://go.dev/wiki/SettingGOPATH

3. Install dependencies

```bash
$ cd gc-backend
$ go get
```

### Frontend

1. Install Node.js (18.13.0+)

2. Install dependencies

```bash
$ cd gc-frontend
$ npm run install
```

## Run in dev mode

### Inside gc-backend folder
```bash
$ go run .
```

### Inside gc-frontend folder
```bash
$ npm run start
```
